## 3. 评分逐项理由

### O1 AI 职业赚钱路径诊断网页 — 8.57 ⭐

- 痛点 8：用户不是没方案，是选择困难。中。
- 付费 6：v0.1 免费，付费要通过后续漏斗。
- 增长 9：AI 话题天然热；输出可分享。
- MVP 速度 10：纯前端，Claude Code 1–2 次会话可出 v0.1。
- 差异化 8：市场上没有明确同类产品（见 `competitor_map.md`）；但如果输出"套话"差异化会垮，要靠 PRD 保证。
- 自动化 9：规则化模板生成，无实时 API 调用也能跑。
- 个人匹配 10：纯前端 + Markdown 输出 = Claude Code + GitHub Pages 完美贴合。

### O2 Prompt 模板库 — 7.86

- 天然的 O1 下游变现。但单独起步没有传播入口。建议作为 O1 的**第一个付费升级**，不单独立项。

### O3 AI 转型定制报告 — 7.43

- 高付费意愿，但需要 O1 作为免费漏斗。单独做没流量，做成 O1 的"高价档"最合理。

### O4 AI 职业转型课程 — 5.86

- 生产周期长（≥2 周）；同质竞争严重；录课对个人创作者成本高。**不做**。

### O5 1v1 咨询 — 6.43

- 最高客单价但不适合 AI 自动化（自动化 2）。需要人力。可作为 O1 金字塔最顶端延伸，不作为项目主轴。

### O6 通用 App — 5.29

- MVP 速度 3：登录/数据库/多职业覆盖，重产品。**与 v0.1 原则冲突，否决**。

### O7 Newsletter+社群 — 5.71

- 需要至少 6 个月内容积累才能形成飞轮。短期无法验证。**不做**。

## 4. 策略取舍（关键决定）

### 做什么

- 🎯 **第一优先：O1 诊断网页（v0.1）**
- 🧭 **路线图：O1 → 附带 O2 单职业模板包作为低价付费 → O3 高价定制为金字塔顶端**

### 不做什么（明确写出）

- ❌ 不做体系课（O4）
- ❌ 不做通用 App（O6）
- ❌ 不做 Newsletter/社群（O7）
- ❌ v0.1 不独立做 O2/O3，等 O1 验证后再接

### 假设与失败条件

- O1 v0.1 上线 30 天内如果无任何主动转发或收藏 → 回到 Gate 1 重审用户画像
- 如果有用户反馈"输出像套话" ≥ 50% → 回到 Gate 3 重做输出结构，不要硬推

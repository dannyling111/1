# Page Structure · AI 职业赚钱路径诊断器 v0.1

**Gate**：G3
**单页**：只有一个 `index.html`，由 5 个视觉区块组成。

---

## 区块 1：Hero

```
AI 时代，你的职业还怎么赚钱？
填 9 个问题，5 分钟，拿到你的专属赚钱路径与 30 天行动清单。
[ 开始诊断 ↓ ]
```

约束：
- 标题 ≤20 字
- 副标题 ≤40 字
- 一个主 CTA 按钮，跳锚到表单

---

## 区块 2：说明（极短）

```
不是套话。
报告会引用你填的职业、技能、时间、风险偏好，给出具体到"明天就能做的事"的清单。
```

约束：
- 2 行以内
- 不要列"为什么选我们"、"4 大优势"之类的套话

---

## 区块 3：表单

分 3 组视觉分区（不是分步），一个页面内：

```
▍关于你
  1. 你现在做什么？        [下拉 + 其他]
  2. 你的核心技能 (≤3)     [tag 多选]

▍你的条件
  3. 每周能投入多少时间？  [单选]
  4. 目标月收入            [单选]
  5. 风险偏好              [单选]

▍你的意愿
  6. 愿意做内容？          [Yes/No]
  7. 会写代码？            [Yes/No]
  8. 愿意做咨询？          [Yes/No]
  9. 愿意做产品？          [Yes/No]

[ 生成我的路径 ]
```

约束：
- 所有字段标签左对齐
- 移动端每字段独占一行
- 桌面端"关于你"和"你的条件"两栏
- tag 使用 `<button>` 切换 selected 状态，不用 `<select multiple>`

---

## 区块 4：报告

生成后出现。结构：

```
┌────────────────────────────────┐
│ 你的 AI 赚钱路径报告           │
│                   [复制 Markdown]│
├────────────────────────────────┤
│ 概览：职业|技能|时间|目标      │
│                                │
│ 1. AI 替代风险：62/100         │
│    <一段说明>                  │
│                                │
│ 2. AI 增强机会（3 条）          │
│                                │
│ 3. 最适合的 3 条赚钱路径        │
│    - 路径 A                    │
│    - 路径 B                    │
│    - 路径 C                    │
│                                │
│ 4. 7 天行动计划                 │
│ 5. 30 天行动计划                │
│ 6. 90 天行动计划                │
│ 7. 推荐工具栈                   │
│ 8. 下一步：3 件明天就能做的事    │
└────────────────────────────────┘
```

约束：
- 报告区在表单下方，生成时 `scrollIntoView({behavior:'smooth'})`
- 复制按钮固定在报告卡片顶部右侧
- 手机端复制按钮放在 header 内部（不 fixed 遮挡）

---

## 区块 5：Footer

```
本工具 v0.1 · 规则化生成，不调用外部 AI · 不收集个人数据
GitHub: [repo link]  ·  反馈：[email]
```

约束：
- 一行文字
- 不做 newsletter、付费、社群入口（v0.1 不做）

---

## 视觉基调

- 字号：body 16px，h1 28px，h2 22px，h3 18px
- 颜色：浅色背景 `#fafafa`，正文 `#1a1a1a`，CTA `#0066ff` 之类朴素色
- 圆角 8–12px，阴影极淡
- 不引 icon 库（需要时用 emoji 或纯 CSS）

---

## 不做项（强调）

- ❌ 没有 Hero 大图 / 视频
- ❌ 没有 "4 大优势"
- ❌ 没有"合作伙伴 logo 墙"
- ❌ 没有弹窗订阅邮件
- ❌ 没有 cookie 同意（v0.1 不种 cookie）
- ❌ 没有多语言切换按钮
- ❌ 没有暗色模式（v0.1 不做）

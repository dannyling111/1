## 2. 间接竞品

### 2.1 "AI 会替代我吗"测试类

| 产品形态 | 代表 | 输入 | 输出 | 对我们的启示 |
|---|---|---|---|---|
| "Will AI Replace Me" 类网页 | 多个英文小站 | 选择职业 | 替代百分比 | **只给焦虑，不给行动**。我们要给行动。 |
| 职业兴趣测试 | 16Personalities, 霍兰德 | 40+ 题 | 性格/职业匹配 | 输入量太大；我们要压缩到 10 字段。 |

## 3. 竞争格局判断

### 定位象限（自评）

```
            深度/个性化
                 ↑
                 │
    [1v1 咨询]   │
                 │
                 │   [ ← 本项目 v0.1 定位 ]
                 │   (轻量 + 结构化 + 免费 + 可分享)
                 │
    [课程/社群]   │
                 │
    [新闻/工具库]│
                 └────────────────→ 广度/规模化
```

v0.1 目标站位：**比课程个性化，比 1v1 便宜 (免费)，比"替代测试"有用**。

### 直接竞品空缺的原因（假设）

1. 做"个人化 AI 建议"容易被认为需要实时 LLM → 成本焦虑 → 没人做规则化版本
2. 做"通用版"又容易失焦 → 大厂不做，小团队也不知道从哪切
3. 垂直到"某一类人"的 AI 转型工具有商业上限感 → 没人愿意只做 10 万人的工具

**机会**：我们用"规则化 + 垂直画像 + 单页"绕过所有这些理由。v0.1 就不追求规模，追求验证。

## 4. 我们相对竞品的核心差异

| 维度 | 传统 AI 课程/社群 | "会被替代吗"测试 | 1v1 咨询 | **本项目** |
|---|---|---|---|---|
| 输入成本 | 高（报名/上课） | 极低 | 中（预约） | 极低（5 分钟表单） |
| 输出颗粒度 | 体系化但通用 | 一个百分比 | 深度定制 | 结构化、分层、可执行 |
| 可分享性 | 低 | 中 | 无 | 高（Markdown + 结构化） |
| 行动导向 | 弱 | 无 | 强 | 强 |
| 付费门槛 | 中–高 | 0 | 高 | 0（v0.1） |
| 升级路径 | 社群/进阶课 | 无 | 续费 | 模板包 → 定制 |

## 5. 借鉴

- **可分享性**来自"职业兴趣测试" —— 输出本身是社交货币。要在 PRD 中设计"输出截屏友好"。
- **减少输入摩擦**借鉴 Typeform 式分步表单（即便是规则化后端，前端也要轻）。
- **漏斗延伸**借鉴 1v1 咨询 —— 免费诊断 → 付费深化是成熟路径。
